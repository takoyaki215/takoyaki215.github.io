Introduction

This project is for Taiwanese people who lost or stolen a new e-identity card, and they need to show these documents to the government to apply for a new ID card.

What I did...

My main work of this project is UI for iOS and android system. To know the specific steps, I use the situational illustrations to help the user to accomplish the registration.
