Introduction

Due to personal computers are not popular in China, the functions of this app includes online learning videos, homework, and quizzes for trainees to learn the new know-how. Thought watching online video and take note, then do the homework.

I coded the CSS and HTML via the Framework which could connect HTML to revise code of iOS and Android. This was one of the projects of Campuscruiser, and I collaborated with the Shanghai engineering team.

Conception

Because of the project timeline, my supervisor opted to build the app using the ionic framework (https://ionicframework.com/) which allowed for code reuse of builds cross‑platform.  Moreover, to create a favorable impression for users, I used an alpaca as the mascot for the first time page, empty page and error page.

The impact

By using the app, the rate that painter workers know the latest expertness is more than before. Plus, the Nippon painter app solves the problems that there is no personal computer at home.
