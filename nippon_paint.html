Nippon paint

Nippon Paint internal Chinese staff learning APP - UX and UI / 2015

TEAM WORK

This APP was published in China. It includes an online learning video, homework, and a quiz trainee employees. To encompass a more friendly user experience and interaction, I used an alpaca as the mascot for the learning app.
My responsibility in this project was with all of the UI and UX design; moreover, I created the CSS and HTML through "ionic" that connected HTML to iOS and Android. This is one of the projects of campuscruiser, and I had to collaborate with the Shanghai engineering team.
