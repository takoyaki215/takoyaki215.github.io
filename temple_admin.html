Introduction

The project is to create a to-do list, table, form, and data of believer, financial affairs of temple and activities of others temple. Each feature phase of this website is serialized, starting with the design and development by using bootstrap.

I designed the website pages to make them match the logo color scheme- golden and blue colors. My responsibilities range from creating UI/UX guidelines and developing frond-end.

Conception

Because of different sizes of the users’ screens nowadays, the button on the right upper corner can be tapped and change the website into full-screen mode. Moreover, a hamburger icon can be tapped that expanded or collapse of the screen.
Besides, to avoid confusion, different colors are used for the tables of income (primary color navy) and cost (sub color gold).  Plus, the export excel button is a special function with gradient color so that can be distinguished from normal buttons.
