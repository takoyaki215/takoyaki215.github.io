NCKU Fintech APP

Problem & Solution

This APP enables the students of Taiwan’s National Cheng Kung University (NCKU) to use a digital discount system in affiliated stores. The merchants activate this via scanning the QR code of the user. This project implements a blockchain discount system from which reward points can be redeemed, which attracts student users and merchants to use this APP for transactions.

Introduction

Initially, NCKU provided points for management by students. Accordingly, students need to pay via an app to utilize earned points and conduct transactions. (Accumulation of points is determined via an exchange rate decided by the university administration.) In addition to access via transactions with merchants, login via personal computer can be done to manage accounts. The major tasks of the index are New Order, Quick Pay, Order Mgmt and Trans Setting.
